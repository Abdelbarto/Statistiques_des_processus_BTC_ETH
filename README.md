# Statistiques des Processus — BTC/ETH 2025

Projet de modélisation statistique des crypto-actifs BTC et ETH
sur données horaires 2025.

## Structure

| Dossier | Contenu | Responsable |
|---------|---------|-------------|
| `garch/` | Modélisation GARCH / EGARCH | Abdelbar |
| `evt/` | Théorie des Valeurs Extrêmes (POT, GPD, GB-EVT) | Abdelbar |
| `copulas/` | Modélisation de la dépendance BTC-ETH | Collègue 1 |
| `data/` | Données brutes et transformées | Commun |

## Installation

```bash
pip install arch scipy statsmodels matplotlib pandas numpy
```
